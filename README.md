## Installation

```sh
yarn add --dev --exact \
  eslint \
  eslint-config-prettier \
  eslint-plugin-react \
  eslint-plugin-react-hooks \
  eslint-plugin-react-native \
  @typescript-eslint/eslint-plugin \
  @typescript-eslint/parser \
  @oursky/eslint-config
```
